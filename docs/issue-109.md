## 项目类

### <font color="red">1、programmable_precision_resistor - 一个开源可编程精密电阻仪</font>

该项目是一个纯开源的可编程电阻仪，其电阻值可以通过数字信号来调节，可调范围 1Ω - 999.999KΩ，支持 SCPI 可编程仪器标准命令。

 * 项目地址：https://github.com/sbstnh/programmable_precision_resistor
